CS156: Finding Patterns in Data with Machine Learning
Spring 2026
Course Description
Students learn to apply core machine learning techniques — such as classification, perceptron, neural
networks, support vector machines, hidden Markov models, and nonparametric models of clustering —
as well as fundamental concepts such as feature selection, cross-validation and over-fitting. Students
program machine learning algorithms to make sense of a wide range of data, including images and
numerical data.
Course Objectives & Learning Outcomes
Core machine learning knowledge and skill mastery.
#cs156-MLCode: Produce working, readable, and performant Python implementations of a
variety of machine learning systems using appropriate libraries and software tools.

visualizations.

calculus, and Bayesian statistics.
Produce original machine learning work that goes beyond the scope of what was covered in class.

Unit 0: ML Fundamentals
(v.21. Last updated: 2026-03-23 17:27:48.959822+00:00)

Machine learning stands at the intersection of statistics and computer science. It is a greatly promising
set of ideas that allow us to extract valuable information from the large volume of data that society is
currently producing. In this unit we provide a basic framework we can use to begin to answer some of
the more basic questions: What is a machine-learning algorithm? What data are appropriate for a given
algorithm? How should we prepare the data and then validate the model afterwards? If we have a model,
how do we decide on any actions to take based on the model's output?

Unit 2: High-dimensional Data
"Ignorance is bliss"
– Cypher
This unit introduces high-dimensional data and the challenges this introduces for simplistic ML
approaches. It begins by characterizing overfitting, cross-validation, and the bias-variance trade off. It
continues with the challenges of high-dimensional spaces and how to use linear algebra to estimate
values within them. The unit introduces a number of more complex ML models including SVMs, CNNs,
Random forests, and Autoencoders.

Course Policies and Assessment
Please refer to the following pages on myMinerva regarding general course policies and assessment:
https://my.minerva.edu/academics/policies/course-policies/
https://my.minerva.edu/academics/policies/course-grading-assessment/
For the details regarding the overall course score calculation in this course, please refer to the “Course
Scoring” section of this syllabus, above. The same information is reflected in the Outcome Index on
Forum: from the "Courses" tab, click on "View Course Score Calculation".
Note: There may be minor changes to the syllabus before class launch due to integrating some
new AI assessment tools and course structures. These are unlikely to substantially impact the
content or class policies.
For this course:
1. 25% of the course's grade is in the form of pre-class work books. These are micro-assignments,
not merely in course prep. If the pre-class work was submitted but you were unable to attend class
your PCW will be evaluated as normal and can receive full credit. If your PCW is not submitted
before class, it will receive a 0 even if it is your first or second absence. It is thus important to do
the PCW even on days you can't make it. There no make-up work for the PCW, but each review
day has an opportunity to resubmit one PCW for additional credit.
2. 25% of the course's final grade is based on in-class participation. This is a combination of poll
responses, verbal contributions, and breakouts. However, for bookkeeping reasons, these will all
be integrated into scores attached to a poll on each review day. These days are therefore
particularly important to attend. Please note that because class contributions are part of my overall
assessment of participation, absences and tardiness can impact this score, both due to penalty
scores and due to reduced opportunities to participate.
3. It is highly recommended that you watch the class video for any classes you miss, but there is no
make up work, and you're encouraged to shift to the next PCW and prep for the upcoming class
rather than worrying about what's gone before. In each review class you can re-submit a PCW you
missed to receive partial credit.
4. Assignments turned in more than an hour after the due date will receive a late penalty.
Assignments after the close date will not be accepted.

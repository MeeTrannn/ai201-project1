Course Description
How do weather scientists know for sure that a hurricane will hit the coast in two days? How would one
devise methods to forecast sales of a product? At the end of this course, you will be equipped with tools
to gather and interpret data, and to inform a scientific or business decision on the basis of statistically
rigorous arguments.
We start the course by applying PyMC — a Python library for automatically computing approximate
posterior distributions — to a case study. This allows you to see how we use a state-of-the-art tool to
compute Bayesian posterior distributions in non-trivial statistical models. Next, we develop our statistical
modeling toolbox by learning how to create ever more complicated models for complicated data sets. We
also learn how to identify when a model is too complicated and how to avoid overfitting in Bayesian
inference. The last four weeks of the course are spent on understanding how PyMC works under the
hood — how we can fit functions and draw samples to compute approximations of complicated posterior
distributions.

Resources for refreshing your Probability and Statistics knowledge
If you want a detailed CS114 refresher, work through Statistics 110: Probability by Joe Blitzstein at
Harvard. This page contains links to a free textbook, a free online course, and a cheat sheet. The
course has animations and practice questions embedded but is otherwise similar to the textbook.
So, you should either go through the textbook or take the course to save time. The probability
cheat sheet is handy if you want a quick refresher.
The Statistics Fundamentals YouTube channel by Josh Starmer (StatQuest). Easy-to-understand
explanations with clear visualizations.
Statistics and Probability by Khan Academy. It is a good resource for students who feel they lack
foundational probability knowledge or are relatively new to probability. There are lots of exercises


available.
For a more technical background reading on basic probability and statistics, see MacKay’s
Information theory, inference, and learning algorithms, Chapter 2 (available for free at
http://www.inference.org.uk/itila/book.html) up to the end of Section 2.3. Do all examples and
exercises in MacKay’s text up to the end of Section 2.3 — there are 11 of them. Optionally, you can
also do Exercises 2.16, 2.21–2.24, 2.31, 2.33, 2.34, 2.36, 2.37. If you are really into statistical data
modeling, work through everything in Chapter 2.
Scripting in Python
Familiarity with Python is essential for computational statistical modeling in CS146. You are not required
to write complex Python programs in this course, so no knowledge of object-oriented programming,
programming best practices, or software engineering is assumed.
We use Python 3 and a package called PyMC in this course. Pre-class work can be done on Forum
where the required packages are installed for you. If you want to run code on your own computer, the
recommended Python 3 distribution is Anaconda, which you can download for free. You may also use
cloud-based Python notebooks — DeepNote and Google Colab both support PyMC. Sign up with your
Minerva email address to get free educational access.
You should know how to write functions, for-loops, while-loops, conditional (if-elif-else) statements.
You should be comfortable with manipulating basic data structures like classes, lists, dictionaries,
and strings,
You should have a passing familiarity with NumPy and SciPy.
You should have a passing familiarity with plotting data using Matplotlib.

Unit 2: Automated inference with PyMC
Doing Bayesian inference by hand is tedious and error-prone — especially in complicated models.
Modern statistical models are solved using automated methods. However, none of these automated
methods are perfect — they all make assumptions and approximations. In practice, we need to be aware
of how these automated methods can fail and how to diagnose and fix problems as they come up.

Unit 4: Monte Carlo inference algorithms
Monte Carlo methods use random samples to replace intractable integrals with straightforward sums,
allowing us to approximate any complex model or distribution. We learn how to implement and discuss
the convergence properties of a few simple Monte Carlo methods before we get a conceptual
understanding of how an advanced sampling algorithm used in PyMC works.

Unit 5: Variational inference algorithms
Variational algorithms approximate a complicated probability density function using a simpler density
function, allowing us to compute approximate expected values and make approximate predictions for a
complicated statistical model. We learn how simple variational approximations are computed before we
get a conceptual understanding of how an advanced variational inference algorithm in PyMC works.

Overview
There is no make-up work in CS146, but you must always submit pre-class work, even if you are
going to be absent.


Pre-class work
The typical class format of CS146 places a strong emphasis on pre-class work. The pre-class work
almost always forms the basis of an activity in class, and you will not learn meaningfully from the class
session if you have not worked through the pre-class exercises in detail.
You have to do the pre-class work every time.
Even if you intend to skip a class, you have to do the pre-class work, and you will still get
graded on it.
You do not have to get everything right, but you have to try, and you have to show your work. If you
get stuck and find that you are unable to complete the pre-class work, explain why. Explain what
you attempted and why you think it didn’t work.
Teaching assistants will provide formative feedback on your pre-class work to help you learn from
your mistakes and improve.
You will frequently get graded on pre-class work as part of your in-class assessment.
Which sessions get graded on pre-class work is determined randomly.
Not completing pre-class work to a satisfactory level will result in a low score.
Not engaging meaningfully with the pre-class work at all will result in a score of 0.
Make-up work
There is no make-up work, but remember that you have to submit your pre-class work even if you
will be absent from class.
It is in your best interest to watch the class recording when it is available. If the video recording is
unavailable for some reason (for example, a technical problem), contact your instructor. Where
possible, a recording of another section of the same course will be shared with you.
